# Monitoring-Haji
Penyelenggaraan ibadah haji merupakan kegiatan nasional berskala besar yang melibatkan ribuan jamaah dari berbagai daerah, termasuk Provinsi Lampung. Dalam praktiknya, diperlukan sistem monitoring yang terintegrasi untuk memastikan keamanan, kesehatan, dan kelancaran ibadah jamaah sejak keberangkatan hingga kepulangan.
